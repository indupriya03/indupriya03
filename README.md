## Featured Projects

### ⚡ [Rapido Ride Intelligence System](https://github.com/indupriya03/Rapido-Ride-Intelligence-System)
> End-to-end ML system for ride-hailing intelligence

Built 4 production-grade ML models on large-scale ride-hailing data:
- **Booking outcome** prediction (multi-class, Weighted F1: 0.749)
- **Fare forecasting** per vehicle type (R² > 0.994 — near-perfect accuracy)
- **Cancellation risk** detection (ROC-AUC: 0.864, threshold-optimized)
- **Driver delay** classification (ROC-AUC: **0.9997** after Optuna tuning)

Served through a unified inference engine and a branded 4-page Streamlit dashboard with dark/light mode, live single-ride predictions, and business strategy recommendations.

`XGBoost` `LightGBM` `Optuna` `SHAP` `SMOTE` `Streamlit` `SQLAlchemy` `MySQL`
